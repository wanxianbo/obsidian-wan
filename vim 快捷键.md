# vim 快捷键

```javascript
shift+g : 页面尾行
gg : 页面首行
ea : 在单词末尾编辑
wi : 在单词最前面编辑
vi + p("'[{ ： 选中最近空行("'[{的内容
di + p("'[{ ： 剪贴最近空行("'[{的内容
yi + p("'[{ ： 复制最近空行("'[{的内容
ci + p("'[{ ： 删除最近空行("'[{的内容
shift+$ : 该行文本末尾
shift+^ : 该行文本首
shift+d : 删除光标之后的所有内容(包含光标的内容)
shift+s : 删除整行，进入编辑模式
shift+i : 行首编辑
shift+a : 行尾编辑
zz : 光标显示编辑框的最中间
shift+v : 进入多行编辑模式
shfit+>> : 往后tab一个空格
shfit+<< : 往前tab一个空格
shift+l : 当前屏幕末尾
shfit+h : 当前屏幕最开始
shift+8 : 查找当前当前光标所在单词
shift+| : 当前行首
shift+% : 对应括号跳转
d + ^ : 删除光标前面的所有元素
:/ : 搜索 n 往后查找 N 往前查找
set (no)hlsearch : 查找(取消)设置高亮 
```
